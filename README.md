# Smart Demand Forecasting and Delivery

Simple end-to-end supply chain starter for Challenge 3 with realistic datasets and a clear pipeline:

DATA (2 years) → PREDICT → PLAN → DELIVER → VISUALIZE

## What this project does

1. **Predict**
   - Forecast tomorrow's demand per store-product using 2 years of historical sales data with growth factor.

2. **Plan**
   - Compare predicted demand against maximum warehouse inventory for each product.
   - Identify shortages (demand > warehouse stock).

3. **Deliver**
   - Assign each store-product pair to the closest warehouse that stocks that product.
   - Calculate transport cost using distance × quantity.

## Project structure

```
data/
  ├── historical_sales.csv    (40,205 records: 2 years × 11 stores × 5 products)
  ├── warehouses.csv          (25 records: 5 warehouses × 5 products)
  ├── distances.csv           (55 records: 5 warehouses × 11 stores)
  └── output.csv              (generated by pipeline)

src/
  ├── pipeline.py             (prediction + planning + delivery logic)
  └── dashboard.py            (Streamlit visualization)
```

## Datasets explained

### 1. historical_sales.csv
**What:** 2 years (2024-2025) of actual store sales by date, store, and product.

**Columns:**
- `date`: sales date (Jan 2024 - Dec 2025)
- `store`: store identifier (A-K = 11 stores)
- `product`: product type (water, milk, soda, cheese, bread)
- `sales`: units sold that day

**Example row:**
```
2024-01-07,A,water,151    # Store A sold 151 units of water on Jan 7
```

### 2. warehouses.csv
**What:** Inventory available at each warehouse for each product.

**Columns:**
- `warehouse`: warehouse name (W1-W5 = 5 specialized warehouses)
- `product`: product type
- `stock`: units currently available

**Example row:**
```
W1,water,5000    # Warehouse 1 has 5000 units of water
```

### 3. distances.csv
**What:** Distances (in km) between each warehouse and store.

**Columns:**
- `warehouse`: warehouse name
- `store`: store name
- `distance_km`: distance in kilometers

**Example row:**
```
W1,A,144    # Warehouse 1 is 144 km from Store A
```
